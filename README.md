Build real applications on Avalanche without overthinking setup.

This repository provides a collection of minimal, runnable starter kits designed to help developers quickly understand and build on Avalanche using practical examples.

Each starter focuses on a specific use case and is structured to be easy to run, understand, and extend.

---

## 🚀 What you'll find here

A set of focused starter kits covering common patterns:

- Token contracts and basic interactions  
- NFT minting flows  
- Payments and transfers  
- Contract interaction patterns  
- Event listening and indexing  

Each starter is intentionally simple and avoids unnecessary abstractions.

## 📦 Starter Kits

| Starter | Description | Path |
|--------|------------|------|
| Token Starter | Deploy and interact with an ERC20 token using Core Wallet | [View](./token-starter) |
| NFT Starter | Mint and interact with NFTs (coming soon) | [View](./nft-starter) |

---

## 🔑 Core Wallet

All starters are designed to work with Core Wallet.

Core injects `window.ethereum`, so it works seamlessly with ethers.  
No additional wallet libraries are required.

---

## 🧱 How to use this repo

1. Clone the repository
2. Navigate to any starter folder
3. Install dependencies
4. Run the project

Each starter contains its own setup instructions.

---

## 🎯 Design Principles

This repo is built around a few simple ideas:

- Minimal → no unnecessary complexity  
- Practical → real use cases only  
- Runnable → everything should work out of the box  
- Extendable → easy to build on top of
